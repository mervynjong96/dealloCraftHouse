To use the web application, the database needs to be setup first.
1. The database file is contained inside the database folder of current directory, it includes the database structure and data of this web application.
2. Open up PhpMyAdmin in your web browser and directly click on "Import" button.
3. Select the database.sql that located inside the database folder and directly click "Go" button to import all the data structure and data.
4. That is all!
